# 文件说明
.gba为gba游戏文件本体
.sav为玩家1(默认)的gba游戏存档(这个不会因为模拟器的不同而不同)
.sa2为玩家2的gba游戏存档(mGBA模拟器特有,其实就是.sav文件)
.ss[number]为玩家1的即时存档[number]的文件
上述的.gba/.sav/.sa2/.ss[number]文件默认会在同一个文件目录下
.sav/.sa2/.ss[number]为mGBA模拟器生成的存档文件
而且文件名都相同，都是[name].gba文件的文件名[name]
# 本地联机说明
## 事前准备
拷贝[name].sav为[name].sa2
这个是因为新建多人游戏窗口之后，玩家2的游戏存档默认是空的，也就是从头开始打
这显然不符合我们的需求，于是直接强制拷贝玩家1的游戏存档给玩家2用
## 开始联机
在游戏打开的过程中，选择[文件]>>[新建多人游戏窗口]
之后会打开一个新的mGBA模拟器窗口，选择[文件]>>[加载游戏ROM]，
选择[name].gba的本地文件路径，
然后mGBA模拟器就会自动导入和游戏文件同一目录的.sa2文件作为玩家2的游戏存档，
至此，就可以开始联机了，选择游戏里面的自带的有线联机即可。
